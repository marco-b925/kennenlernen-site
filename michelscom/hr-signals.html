Setup in 5 Schritten

Backend-Abhängigkeiten
cd backend · python -m venv venv · source venv/bin/activate · pip install -r requirements.txt
BA-API konfigurieren
.env-Datei im Backend-Root anlegen:
JWT_SECRET=your-secret-key
BA_API_BASE=https://rest.arbeitsagentur.de/jobboerse-jobsearch/pvk-shared-public-search-api
DATABASE_URL=sqlite:///./hr_signals.db
Backend starten
cd backend · source venv/bin/activate · uvicorn main:app --reload --port 8000
Frontend starten
cd frontend · npm install · npm run dev
App testen
Öffne http://localhost:5173 im Browser. Registrieren, einloggen, Job-Suche starten.

Was die Plattform kann

User registriert sich → sucht Unternehmen + Keywords → Plattform durchsucht 6M BA-Jobanzeigen → zeigt Firmen mit aktuellem Recruiting-Bedarf → → Export als Excel (OpenPyXL)

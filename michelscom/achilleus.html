Achilleus

Second Brain Bot — Telegram-Bot, der Sprachnachrichten, Fotos, Dokumente und Text durch Claude Sonnet zu Craft MCP routet

Architektur

Eingabe-Routing

🎤 Voice Memo

Sprachnachricht → ElevenLabs Speech-to-Text → Claude analysiert Inhalt → erstellt Task/Dokument in Craft oder antwortet direkt.

📷 Photo

Foto/Screenshot → Claude Sonnet Vision extrahiert Text & Kontext → kategorisiert als Referenzmaterial → speichert in Craft-Ordner.

📄 Document

PDF/PPTX/DOCX → Text-Extraktion → Claude fasst zusammen & leitet Aktionen ab → strukturierte Notiz in Craft.

💬 Text

Direktnachricht → Claude klassifiziert (Task/Idee/Notiz/Frage) → routet an passenden Craft-Bereich. Memory hält 20 Nachrichten Kontext.

System Prompt

Der Agent arbeitet mit diesem Prompt — er entscheidet autonom, was wohin in Craft geschrieben wird:

Du bist Achilleus, Marcos Second Brain Bot via Telegram.

Deine Aufgabe: Eingehende Nachrichten (Voice, Foto, Dokument, Text)
kategorisieren und strukturiert in Craft ablegen.

REGELN:
1. Voice → transkribieren, dann wie Text behandeln
2. Foto → Vision-Analyse, Kontext extrahieren
3. Dokument → Inhalt zusammenfassen, Kernpunkte destillieren
4. Text → Klassifizieren als: Task / Notiz / Idee / Frage

CRAFT-ROUTING:
- Task → craft://tasks (mit Fälligkeitsdatum falls erkennbar)
- Notiz → craft://docs/marco-inbox (mit Kategorie-Tag)
- Idee → craft://docs/ideen-parkinglot (ungefiltert, später reviewen)
- Frage → Direkt beantworten, NICHT in Craft speichern

ANTWORT-STIL: Kurz, direkt, ohne Floskeln. Immer bestätigen,
WAS du wohin geschrieben hast (z.B. "✓ Task in Craft angelegt").

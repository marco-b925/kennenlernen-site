## Projekt: Link (Odysseus-Fork)

- **Repo:** https://github.com/marco-b925/link
- **Beschreibung:** Fork von "Link" — ein selbst-gehostetes LLM-Chat-Frontend mit umfangreicher Tool-Integration. Lokal betrieben (localhost:7860), verbindet sich mit verschiedenen LLM-APIs.

### Tech-Stack
- Python-basiert (Gradio Framework)
- Multi-Model-Support via API-Integrationen
- Datei-Upload + Drag-and-Drop in Chat
- Lokale Installation mit Python-Abhaengigkeiten

### Features
- **New Chat + Search:** Unterhaltungen durchsuchen und neue starten
- **Agent/Chat-Modi:** Umschaltbar zwischen autonomem Agenten und einfachem Chat
- **Multi-Model:** Verschiedene Modelle parallel nutzbar (MiniMax M3, Claude, GPT etc.)
- **Integrierte Tools:** Email, Brain (Wissensdatenbank), Calendar, Compare (Modell-Vergleiche), Cookbook (Prompt-Vorlagen), Deep Research, Gallery, Library, Notes, Tasks
- **Theme-Support:** Anpassbares UI-Theme
- **File-Drop:** Dateien direkt im Chat verarbeiten

### Warum relevant fuer diese Bewerbung
Zeigt tiefes Verstaendnis des LLM-Oekosystems: API-Integration, lokale LLM-Infrastruktur, Tool-Orchestrierung. Fork beweist dass ich bestehende Open-Source-Tools evaluieren, forken und fuer eigene Zwecke anpassen kann — eine Kernkompetenz fuer die Rolle.
